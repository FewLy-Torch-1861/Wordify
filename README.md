# Wordify
Just a text encode/encryption/hash tool for Arch-based Linux written in Python.

## Supported encryption type
- Caesar Cipher
- Reverse

## Supported encoding type
- Base64
- Base32
- Base16

## Supported hash type
- SHA256

# Installing
```sh
git clone https://github.com/FewLy-Torch-1861/Wordify.git
cd Wordify
./setup.sh
```
